# VulkanTemplate
Base Vulkan project setup with cmake

## How to start
If You are using just file simpliy type `j b r` and this should pop out empty window
If not then standard cmake approach will work

## Dependencies

  - glfw3 
  - vulkan 
  - dl 
  - pthread 
  - Xxf86vm 
  - Xrandr 
  - Xi 
  - fmt
